Project 1

Restaurant Billing System

The Restaurant Billing System is a simple application designed to manage customer orders and generate bills efficiently. It allows the user to add food items, calculate the total amount, include taxes, and display the final bill instantly. The system helps restaurant staff save time, reduce manual errors, and maintain accurate billing records. It was built using the MERN stack (MongoDB, Express, React, Node.js) with real-time features enabled by Socket.IO.

Project 2

Student Recognition System

The Student Recognition System is an advanced application that identifies students using a webcam and displays their information instantly. When a student comes in front of the camera, the system recognizes their face and retrieves details such as name, class, roll number, section, and address from a database. This project helps in automating attendance, monitoring student presence, and managing student records efficiently, demonstrating expertise in Computer Vision and MERN stack technologies.

Project 3

Hand Gesture-Based Hill Climbing Robot Control

This project is an innovative system that allows a robot to climb hills or uneven terrain controlled entirely through hand gestures. Using a webcam or camera sensor, the system detects and interprets specific hand movements, translating them into commands for the robot. It leverages computer vision and advanced gesture recognition algorithms to provide a seamless and intuitive control experience, combining AI and Robotics principles.
